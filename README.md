## 💻 Instalasi

1. **Clone Repository**
   ```bash
   git clone https://github.com/Zakvsl/RS-Bunda-Website.git
   ```

2. **Buka Project**
   ```bash
   cd RS-Bunda-Website
   ```

3. **Jalankan Website**
   - Buka file `index.html` di browser
   - Atau gunakan live server VS Code
